# Portfolio Website

A modern, production-ready portfolio built with Next.js, TypeScript, and TailwindCSS.

## Overview

This project demonstrates my approach to building scalable, performant, and user-focused front-end applications.

The site is structured using reusable components and follows modern best practices for layout, styling, and performance.

## Tech Stack

- Next.js (App Router)
- TypeScript
- TailwindCSS
- React
- Vercel (deployment)

## Features

- Component-based architecture
- Responsive design
- Modern UI with clean typography and spacing
- Optimized font loading using next/font
- Structured layout with reusable sections

## Sections

- Hero (value proposition)
- Skills / Focus
- Experience
- Projects
- Global header and footer

## Purpose

The goal of this project is to showcase:
- front-end engineering fundamentals
- component architecture
- performance-aware development
- clean UI implementation
